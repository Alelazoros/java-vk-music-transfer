vk-music-transfer
=================

?

Requirements
------------

- Java >= 8.0

Requirements for developing
---------------------------

- Gradle >= 4.3

Run
---

### For developing


```bash
$ ./gradlew run
```

### For using

```bash
$ java -jar vk-music-transfer.jar
```

Build
-----

```bash
$ ./gradlew build
```


Run tests
---------

```bash
$ ./gradlew test
```
